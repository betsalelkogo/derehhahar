# דרך ההר – דף נחיתה

דף נחיתה בעברית (RTL) לטיולי ריידרים בהר חברון, בהשראת [הר ריידר](https://www.r-rider.info/).

## הפעלה מקומית

```bash
cd /Users/bkoginsky/Desktop/test_tx_analayzer
python3 -m http.server 8080
```

פתחו בדפדפן: http://localhost:8080

## פריסה

העלו את כל הקבצים (כולל תיקיית `images/`) לשרת האחסון של שם המתחם העברי שלכם.

## התאמות מומלצות

1. **טלפון** – הוסיפו מספר ב-`index.html` (כותרת ו-CTA) כשיהיה זמין.
2. **טופס** – כרגע הטופס פותח `mailto:`; לקבלת פניות אוטומטית השתמשו ב-Formspree, Netlify Forms או שרת משלכם.
3. **כתובת** – עדכנו את שורת "אזור פעילות" אם יש כתובת מדויקת למפגש.

## מבנה

- `index.html` – תוכן בעברית
- `styles.css` – עיצוב לפי לוגו (סגול, חרדל, קרם)
- `script.js` – טופס וגלילה
- `images/` – לוגו, גיבור וגלריה
